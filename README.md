SS Jewellers

A Django-based e-commerce web application for a jewellery store — SS Jewellers. The platform allows users to browse products, view catalogues, and contact the store, with user authentication built in.


🚀 Features


🏠 Home page
💎 Product listing page
📖 About Us page
📋 Catalogue section
📞 Contact Us page
🔐 User Authentication (Login / Sign Up / Logout)
📱 Fully responsive design using Bootstrap 5
🗄️ SQLite3 database



🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| Python / Django | Backend framework |
| SQLite3 | Database |
| Bootstrap 5 | Frontend styling |
| HTML / CSS | Templates & custom styles |
